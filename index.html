Starter project

Ce projet contient le minimum syndical afin de commencer un petit projet front-end dans les meilleurs conditions possibles.

Contenu

  • Un compilateur SCSS
  • Un compilateur JS
  • Un dossier pour vos images et icones (Contiendra de quoi minifier tout ça WIP)
  • Le live reload

Si la commande npm run start à chaque sauvegarde, le CSS et le JS sont minifiés et envoyés dans le dossier dist.

Après avoir pris connaissances de ça, vous pouvez supprimer le contenu de ce fichier et du fichier styles.scss et commencer votre projet ou vos tests.
